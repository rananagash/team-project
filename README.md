# MovieNight

### Software Design Project - CSC207 Fall 2025

## Project Summary
**MovieNight** is a Java Swing desktop application that allows users to discover movies, build watchlists, 
record their viewing history, and plan movie nights with friends.
It uses the [TMDb API](https://developer.themoviedb.org/) to retrieve movie information including posters, genres, 
and summaries.

The app is structured using Clean Architecture and SOLID design principles.
